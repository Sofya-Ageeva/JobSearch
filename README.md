# JobSearch - Поиск вакансий

Программа для поиска, хранения и анализа вакансий с сайта hh.ru с использованием базы данных PostgreSQL.

## Функционал

- Ввод названия компании для поиска вакансий.
- Получение данных о вакансиях через API hh.ru.
- Сохранение информации о компаниях и вакансиях в PostgreSQL.
- Фильтрация вакансий по ключевым словам.
- Вывод компаний и количества вакансий у каждой.
- Расчёт средней заработной платы.
- Поиск вакансий с зарплатой выше средней.
- Сохранение данных в JSON (запасной вариант).

## Установка
1. Клонируйте репозиторий:

```
git clone https://github.com/Sofya-Ageeva/AirplaneTracker.git
```

2. Установите зависимости:
```
pip install requirements.txt
```
3. Заполните файл .env в соответствии с шаблоном (.env.template)

4. Настройка базы данных

Для работы программы необходима база данных PostgreSQL.
Запуск PostgreSQL:
```
psql -U postgres
```
Далее необходимо ввести свой пароль для пользователя "postgres"

5. Запуск приложения
```
python main.py
```
